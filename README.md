[Tagober-Flask](https://github.com/doogunwo/TagoBer_Flask.git) 타고버 프로젝트의 머신러닝 애플리케이션 제어용 보조 서버입니다.<br>
[memo](https://github.com/doogunwo/memo.git) 안드로이드 스튜디오에 대한 기본을 학습하기 위해 진행된 Java 프로젝트입니다.<br>
